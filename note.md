# Go言語

- GoはGoogleが開発したオープンソースのプログラミング言語
- 誰が書いても同じようなコードになるのでチーム開発でGood
- 並行並列処理がGoだと簡単に書ける
- クロスコンパイルに対応
  - MacでLinux用バイナリ、Windows用バイナリを作れる
- Go周辺のツールも優れている
- 標準パッケージが超充実
  - 暗号など
- VS Codeがおすすめ。Go Team公式がサポートしている拡張機能が強力
  - GoのLanguage Serverが整形などをしてくれる

## Goのバージョン管理について

- Goは後方互換性を重視（Javaも）
  - バージョン管理ツールを入れなくても最新版だけ入っていれば古いコードも大体動く
  - 複数バージョンが必要な場合は以下で別バージョンをインストールできる
    `go install golang.org/dl/go1.10.7@latest`
